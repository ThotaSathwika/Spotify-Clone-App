# 🎵 Spotify Clone App

A modern and responsive Spotify Clone built to replicate the immersive experience of a real music streaming platform. This project showcases sleek UI/UX design, dynamic music sections, responsive layouts, and smooth navigation.

---

## 🚀 Features

- 🎧 Spotify-inspired UI
- 📱 Fully Responsive Design
- 🎵 Music Cards & Playlists
- 🔍 Search Bar Interface
- 📂 Categorized Music Sections
- 🖱️ Interactive Hover Effects
- 🌙 Dark Theme Interface
- ⚡ Smooth Navigation Experience

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- React.js 
- Tailwind CSS

## ⚙️ Installation

# Clone the repository
git clone https://github.com/ThotaSathwika/spotify-clone.git

# Navigate into project folder
cd spotify-clone

# Install dependencies
npm install

# Start development server
npm start
